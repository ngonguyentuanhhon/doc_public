Review Migrate EDU

I. Mục tiêu & phạm vi

1. Mục tiêu

- Chuyển đổi toàn bộ dữ liệu học tập từ hệt hống edu.vtc.vn ( Moodle 4.2 ) sang hệ thống mới EduCore ( microservice )
- Đảm bảo:
	+ Không mất dữ liệu quan trọng.
	+ Giữ được dữ liệu khóa học, tài khoản, dữ liệu học tập ( hoàn thành khóa học, bài học, điểm, chứng chỉ )

2. Phạm vi

- Bao gồm:
	+ Tài khoản người dùng
		+ Tài khoản học viên
		+ Mật khẩu 
		+ Nhóm/khóa học viên
		+ chức danh

	+ Khóa học: 
		+ Lĩnh vực khóa học
		+ khóa học 
		+ Bài học 
		+ Phôi chứng chỉ 
		+ Cấu hình hoàn thành: khóa học, bài học, chứng chỉ

	+ Dữ liệu học tập:
		+ đăng ký khóa học của học viên
		+ bài học: điểm , thông tin hoàn thành
		+ khóa học: điểm , thông tin hoàn thành 
		+ chứng chỉ khóa học

- Không bao gồm (nếu có):
	+ Log hệ thống cũ

III. Plan

1. Tài khoản người dùng: 
	
	Edu Moodle : cohort  -> user_info.parent_department -> department  -> user
	Edu Core   : Tenants -> Groups                      -> JobTitles   -> Customers

	- Tổng số: > 353.418 account active
		-> SELECT count(id) FROM `mdl_user` WHERE `confirmed` = '1' AND `deleted` = '0' AND `suspended` = '0';

	- [EduMoodle].cohort -> [EduCore.TenantService].Tenants
		-> [cohort] trong moodle sẽ được migrate thành [Tenants] trong EduCore

	- [EduMoodle].user_info.parent_department -> [EduCore.CustomerService].Groups 
		-> [parent_department] trong moodle sẽ được migrate thành [Groups] trong EduCore

	- [EduMoodle].user.department -> [EduCore.CustomerService].JobTitles
		-> [department] trong moodle sẽ được migrate thành [JobTitles] trong EduCore

	- [EduMoodle].user -> [EduCore.CustomerService].Customers
		-> [user] trong moodle sẽ được migrate thành [Customers] trong EduCore

	- [EduMoodle].user.password -> [EduCore.CustomerService].Customers.Password
		+ Password hashing functions được PHP Core cung cấp:
			++ Thuật toán: PASSWORD_BCRYPT, PASSWORD_ARGON2I, PASSWORD_ARGON2ID
			++ moodle đang sử dụng PASSWORD_DEFAULT: từ php > 5.5 => sẽ là PASSWORD_DEFAULT = PASSWORD_BCRYPT
			++ php doc: https://www.php.net/manual/en/function.password-hash.php
		+ Password verify functions được PHP Core cung cấp:
			++ function sẽ dựa vào các ký tự đầu có format như: $2y$10$4........ để phân biệt mà verify theo thuật toán nào
			++ php doc: https://www.php.net/manual/en/function.password-verify.php
